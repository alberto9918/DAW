Menú Principal de la BIOS

El sistema de entrada/salida básico (BIOS) es un programa que controla la comunicación entre todos los dispositivos de entrada y salida del sistema (como, por ejemplo, las unidades de disco, la pantalla, el teclado, el mouse y la impresora). Generalmente, el BIOS se encuentra en un chip de memoria de sólo lectura (ROM) en la placa base del equipo.

El sistema BIOS del equipo tiene una utilidad de configuración incorporada que permite que los usuarios controlen funciones especiales del equipo. Entre la información de configuración almacenada en el BIOS se incluye el tipo de dispositivos periféricos instalados, la secuencia de inicio del equipo y la cantidad de memoria del sistema y memoria ampliada. Cuando el sistema está apagado, una batería de la placa base conserva la información de configuración del BIOS.

Navegación por la BIOS

Moverse por la BIOS
Tecla del teclado Descripción de la función
F1 o F10 Entre a la Configuración en el logotipo rojo de Compaq o el logotipo azul de HP
F1 Ayuda para la configuración
Esc Salir de la configuración
Tecla de flecha hacia arriba o hacia abajo Seleccionar un elemento (mueve el cursor hacia arriba o hacia abajo)
Tecla de flecha hacia la izquierda o hacia la derecha Seleccionar una pantalla de menú distinta (mueve el cursor hacia la izquierda o la derecha)
+ o - Cambiar los valores del elemento seleccionado
Intro Seleccionar submenú
F5 Cargar los valores predeterminados de configuración
F10 Guardar los cambios y salir

Acceso al menú de Configuración del BIOS

Hay por lo menos tres formas diferentes de acceder a los menús de configuración del BIOS, dependiendo del tipo de CPU, el BIOS, y la fecha de fabricación de la notebook. Para la mayoría de los modelos, después de pulsar el botón de encendido para encender el equipo, se muestra un indicación para especificar qué tecla pulsar para entrar en el BIOS o en las funciones de Windows o de diagnóstico. Para los equipos nuevos, será la tecla F10 o Esc . Sin embargo, en equipos muy antiguos, se le puede pedir que presione la tecla Supr para ingresar a la función de Configuración. Si no se pulsa la tecla sugerida en el momento adecuado y el equipo continúa el proceso de arranque, tendrá que apagar el equipo e intentar el proceso de arranque de nuevo.

Pre-UEFI, antes de 2008 para la mayoría de los modelos.
Con el equipo apagado, presione el botón de encendido para encenderlo y pulse varias veces la tecla F10 . Normalmente, la pantalla de bienvenida de Compaq o HP mostrará un mensaje para que pulse F10 para entrar en el BIOS justo antes de que se muestre el menú del BIOS.
Con UEFI, 2008 a mediados de 2011 para la mayoría de los modelos.
Si el equipo tiene un entorno UEFI pre-BIOS, cuando aparece la pantalla de bienvenida HP o Compaq, se le pedirá que presione Escape para ingresar al menú de Inicio. Luego aparece un mensaje para que presione F10 para acceder a la Configuración del BIOS.
Inicio más corto sin pantalla de bienvenida, mediados de 2011 o posteriormente para la mayoría de los modelos.
Para ayudar a los equipos a iniciarse más rápido, en algunos modelos la pantalla de bienvenida de Compaq o HP no se muestra durante el proceso de arranque. En este caso, después de pulsar el botón de encendido, pulse varias veces la tecla Esc para entrar en el menú de inicio, y siga las indicaciones para presionar F10 e ingresar en la Configuración del BIOS.

Opciones del menú principal de la BIOS

Standard CMOS Setup

Desde este menú se puede configurar la fecha y hora del equipo, y las unidades de disco duro y disquete conectadas al ordenador.

También podemos ver la cantidad de memoria instalada.

BIOS Features Setup

Desde aquí se configura la secuencia de arranque del ordenador, el estado del bloqueo numérico, control de acceso al disco duro y disquetes, elegir una contraseña cada vez que arranquemos el equipo o para entrar en el SETUP o si el sistema, al iniciarse, debe llevar a cabo el autochequeo opción Quick Boot.

Chipset Features Setup

Son parámetros que afectan a la RAM y a veces al micro, a los buses AGP, PCI e ISA, y a otros dispositivos, como los puertos serie o paralelo. Puede ser muy peligroso tocar estos parámetros.

Power Management Setup

En este menú se configuran características de ahorro de energía.

PNP/PCI Configuration

Desde aquí se puede determinar si las tarjetas de expansión deben configurarse desde la BIOS o desde el sistema operativo.

Load BIOS Defaults y Load Setup Defaults

Carga valores por defecto de la CMOS.

Integrated Peripherals

Desde aquí se pueden configurar todos los puertos y dispositivos que forman parte de la placa base. Por ejemplo, se puede desactivar la unidad de disquete.

Hardware Monitor Setup

Se configuran parámetros que afectan al monitor.

Supervisor Password y User Password

Permite definir una contraseña de supervisor y otra de usuario. El supervisor tiene acceso a todas las opciones del menú.

IDE HDD AUTO DETECTION

Permite detectar automáticamente los discos duros instalados en el equipo.

Save & Exit Setup

Se graban las modificaciones que se han hecho y se sale de la BIOS.

Exit Without Saving

Se sale de la BIOS sin grabar las modificaciones.Antes de hacer modificaciones sobre la BIOS, hemos de anotar en un papel los valores anteriores de los parámetros.La BIOS Phoenix presenta un menú diferente. Al entrar en la BIOS se accede al menú Main.

Las opciones de menú que aparecen son las siguientes:

Main
Desde este menú se puede configurar la fecha y hora del equipo, y las unidades de disco duro y disquete conectadas, el teclado numérico, las contraseñas de usuario y supervisor; aparece también la cantidad de memoria instalada en la máquina.
Advanced
Aparecen características del microprocesador.Desde aquí se pueden configurar las conexiones serie y paralelo de la placa.
Security
Desde esta opción podemos configurar una contraseña para acceder al Setup y para impedir el acceso al sistema operativo.
Power
Desde aquí se puede activar el modo de ahorro de energía del ordenador o parámetros que afectan al monitor.
Boot
Desde este menú se configura el orden de acceso a las unidades para arrancar el sistema operativo o si el sistema al iniciarse debe llevar a cabo el autochequeo (opción Quick Power On Self Test).
Exit
Nos permitirá salir de la BIOS guardando o descartando los cambios efectuados.
